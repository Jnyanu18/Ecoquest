# **App Name**: EcoQuest

## Core Features:

- User & Role Management: Secure user registration, login, and distinct dashboards for Student, Teacher, and School Admin roles.
- Environmental Learning Modules: Deliver topic-based courses with interactive quizzes to educate students about environmental sustainability.
- AI Learning Assistant: Utilize the Gemini API as a tool to generate concise summaries for learning modules and intelligently create relevant quizzes from provided topics.
- Eco Task Submission System: Students can browse daily eco-tasks, submit photo proofs and textual descriptions, and receive points upon verification.
- Admin Task Verification: School administrators can review, approve, or reject student eco-task submissions, providing feedback when necessary.
- Personal Gamification & Progress: Track student experience points (XP), levels, and display individual progress through completed challenges and learning modules.
- Personal Impact Overview: Provide students with a dashboard summarising their completed tasks, accumulated points, and contribution indicators.

## Style Guidelines:

- Primary color: Deep earthy green (#3B5C23), symbolizing growth, nature, and sustainability within a dark theme.
- Background color: Very dark, desaturated green-grey (#1D231A), providing a sophisticated, eco-conscious foundation for the dark theme.
- Accent color: Vibrant leafy green (#6AED5E), to highlight interactive elements, progress, and provide visual excitement in the gamified interface.
- Headline font: 'Space Grotesk' (sans-serif), for a modern, techy, and gamified feel in titles and headers.
- Body font: 'Inter' (sans-serif), ensuring readability and a neutral, professional look for educational content and task descriptions.
- Utilize line-art or filled icons with rounded corners, themed around nature, achievements, and educational concepts to enhance the gamified eco-friendly experience.
- Adopt a clean, modern card-based layout, optimizing for content clarity, easy navigation, and mobile responsiveness across all dashboards.
- Incorporate subtle animations and transitions, particularly for level-ups, badge achievements, and task confirmations, using Framer Motion for a dynamic user experience.
